关于我们

Girls Coding Day 是一家公益编程体验工作坊，由社会企业 Coding Girls Club发起， 联合众多科技公司和程序员共同举办，致力于促进性别平等，目标是帮助更多的女性进入 STEM领域，架起女性与编程的桥梁。
过去的一年里，Girls Coding Day活动始于武汉，东至上海南京杭州，南到广州深圳，北上北京大连，在9个城市举办10场工作坊，为超过1000名女性带来新鲜的编程体验。
现在，Girls Coding Day从城市走入校园，我们希望：
1、从校园开始，打破刻板印象，鼓励更多大学女学生进入科技领域，带给她们多元性的可能和改变;
2、架接企业与校园，落地执行"100家科技企业支持GCD在校园"活动，推动更多性别友好的科技企业的出现。





志愿者招募

志愿者招募要求

你是我们正在寻找的人吗？

1、帮助女性编程初学者学习编程技能，并进行各类有益的尝试及创新？
2、建立你所在大学的技术人员网络，并成为我们全国志愿者团队中的一员？
3、获得丰富的活动组织经验，并拓展在科技女性领域视野及人脉？
4、通过参与公益组织做很酷的工作，并赚取一定的报酬？
如果您对上述问题的回答是肯定的，那么你可能就是我们正在寻找的人！

你需要做什么？

1、在10月28日，在校内落地启动一个全国性的Girls Coding Day in College的活动；
2、在校内建立Coding Girls Club校园俱乐部，继续承办Girls Coding Workshop（女性编程工作坊）、Girls Coding Camp（女性编程训练营），将这份公益事业继续下去。

我们希望你是怎样的人？

1、认同并愿意传播我们的理念及观点;
2、做事认真踏实，有始有终;
3、富有创造性及公益心，拥有多元化的学习思维;
4、代表 Coding Girls Club具体负责女性编程公益活动在校园的落地;
5、愿意接受 Coding Girls Club提供的相关培训及指导。

我们将提供什么支持？

1、课程体系支持：我们目前已完成多个编程工作坊及训练营课程开发，并将持续开发新的课程内容，为你提供完整的课程方案及前期指导；
2、活动经费支持：我们将提供活动举办必要的经费支持，如有需要，在你寻求本地赞助及合作伙伴时，我们也会提供帮助；
3、技术团队支持：我们将在课程主讲师及教练招募时提供帮助，以实现良好的教学质量；
4、宣传公关支持：我们将通过社交媒体、电子邮件通讯、新闻推广等，提供校园多媒介宣传方案；
5、来自总部团队的持续支持：我们将随时提供各方面的支持，共同推动GCD活动及CGC校园俱乐部的顺利落地。
